Bem-vindo ao Projeto HealthCheck!

Desenvolvimento de uma aplicação moderna com Angular no front-end e ASP.NET Core no back-end.

Estrutura do Projeto

  • HealthCheck.Server: API Web construída com ASP.NET Core.
  • HealthCheck.Client: Interface do usuário desenvolvida em Angular.
  • Cada projeto possui arquivos de configuração e estrutura específicos.

Configuração e Parametrização

Exploramos as configurações principais do ASP.NET Core (como endpoints e serviços) e do Angular (módulos, roteamento e serviços).

Angular CLI em Ação

  • Criação de componentes com boas práticas.
  • Modularização e organização de código.
  • Uso da CLI para automatizar tarefas.

Testes com Jasmine/Karma

Realizamos testes unitários iniciais para garantir a integridade dos nossos componentes.

Conclusão

Próximas entregas: controladores personalizados, roteamento avançado e integração mais profunda entre o front-end e o back-end!
